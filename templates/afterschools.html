After School Clubs

Sharp Sparks offers a range of courses which all run 10 weeks as well as running 3 hr workshops on an as required basis. Three such examples of these are:

  1. Introduction to Programming Wearable Electronics (3hr workshop)

    Twinkle Lily Pad and Sparkle boards are designed to get you started in the world of e-textiles. Students will learn how to program the sparkle board and learn about basic circuits before they turn them into wearable electronics. A small amount of sewing will be required.

  2. Introduction to Computer Programming in Python (10 weeks)

    This course will involve learning basic programming concepts and using these skills to program basic games such as “Magic 8 Ball” and “What’s the number?”.

  3. Introduction to Animation (5 weeks)

    Experiment with a variety of animation styles including cut-out puppets, drawn animation, pixelation, computer animation, flip books and experimental materials. We will learn techniques for bringing any object to life! This club is perfect for first-time animators!
